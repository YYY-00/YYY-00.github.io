# 音频剪辑与音效合成工具

一个基于 Web Audio API 的声音素材剪辑和音效合成工具，完全在浏览器中运行，无需服务器支持。

## 功能特性

### 🎵 音频编辑
- **导入音频** - 支持多种音频格式（MP3, WAV, OGG, FLAC等）
- **波形可视化** - 实时显示音频波形
- **区域选择** - 鼠标拖拽选择音频区域
- **播放控制** - 播放、暂停、停止、播放选中区域

### ✂️ 剪辑工具
- **裁剪** - 保留选中区域，删除其他部分
- **删除** - 删除选中区域
- **复制** - 将选中区域复制到音频末尾
- **淡入/淡出** - 添加平滑的音量过渡
- **音量调节** - 调整整体或选中区域的音量

### 🎹 音效合成
**基础波形：**
- 正弦波 - 纯净的单一频率音
- 方波 - 电子游戏风格的声音
- 锯齿波 - 明亮刺耳的声音
- 白噪声 - 随机噪声（如电视雪花声）
- 粉红噪声 - 自然的背景噪声

**包络音效：**
- 哔声 - 简单的提示音
- Ping音 - 金属感的敲击声
- 底鼓 - 低频打击乐
- 军鼓 - 中频打击乐
- 踩镲 - 高频打击乐

### 💾 导出
- 导出为 WAV 格式

## 使用方法

### 1. 打开工具
直接在浏览器中打开 `index.html` 文件

### 2. 导入音频
- 点击"选择文件"按钮
- 选择一个或多个音频文件
- 音频会自动添加到素材库

### 3. 编辑音频
- 在音频库中点击"编辑"按钮选择音频
- 在波形图上**拖拽鼠标**选择需要编辑的区域
- 使用剪辑工具按钮进行操作：
  - ✂️ 裁剪选中区域
  - 🗑️ 删除选中区域
  - 📋 复制选中区域
  - 🔊 淡入效果
  - 🔉 淡出效果
  - 🔈 调整音量

### 4. 合成音效
- 在右侧参数设置区域调整：
  - 时长（秒）
  - 频率（Hz）- 影响音高
  - 采样率
- 点击任意"生成"按钮创建音效
- 新生成的音效会自动添加到素材库

### 5. 导出
- 选中要导出的音频
- 点击"导出当前音频"按钮
- 文件会自动下载为 WAV 格式

## 技术架构

- **Web Audio API** - 音频处理和播放
- **Canvas API** - 波形可视化
- **Blob API** - 文件导出
- **纯原生 JavaScript** - 无依赖，轻量级

## 浏览器兼容性

支持所有现代浏览器：
- Chrome/Edge 88+
- Firefox 85+
- Safari 14.1+

## 注意事项

- 所有处理都在浏览器本地完成，不会上传到服务器
- 建议使用 Chrome 浏览器以获得最佳性能
- 处理大音频文件时可能需要较多内存
- 导出的音频为未压缩的 WAV 格式

## 快速开始示例

### 生成一个简单的提示音
1. 设置参数：时长 0.2秒，频率 880Hz
2. 点击"生成正弦波"
3. 点击"淡入"添加平滑过渡
4. 点击"导出"

### 制作白噪声背景音
1. 设置参数：时长 10秒
2. 点击"生成白噪声"
3. 点击"淡入"和"淡出"消除爆破音
4. 点击播放试听
5. 满意后导出

### 剪辑音频片段
1. 导入音频文件
2. 在波形图上拖拽选择需要的片段
3. 点击"裁剪选中区域"
4. 导出为 WAV 文件

## 高级技巧

1. **多段音频拼接** - 复制选中区域可以轻松制作循环音效
2. **音量包络** - 先调整音量参数，再添加淡入淡出效果
3. **复合音效** - 生成多个音频，分别导出，然后用其他专业软件合成
4. **频率实验** - 尝试不同的频率值创造不同音效：
   - 20-200 Hz：低音/贝斯
   - 200-2000 Hz：人声范围
   - 2000-8000 Hz：高音乐器
   - 8000+ Hz：高频音效
